---
title: Migration from OpenClaw
description: Migrate your OpenClaw installation and data to SeaClaw.
---

SeaClaw provides a built-in migration tool to transfer your OpenClaw configuration, memory, and sessions.

## Quick migration

```bash
seaclaw migrate openclaw
```

This automatically:

1. Detects your OpenClaw installation directory
2. Converts the configuration to SeaClaw format
3. Migrates memory databases (SQLite, Markdown)
4. Transfers session history
5. Maps provider and channel configurations

## Options

```bash
seaclaw migrate openclaw --source ~/.openclaw     # Custom source path
seaclaw migrate openclaw --dry-run                 # Preview without changes
seaclaw migrate openclaw --skip-memory             # Skip memory migration
seaclaw migrate openclaw --skip-sessions           # Skip session history
```

## What gets migrated

| Component         | Status  | Notes                                              |
| ----------------- | ------- | -------------------------------------------------- |
| Config            | Full    | All settings mapped to SeaClaw equivalents         |
| Providers         | Full    | API keys and endpoints preserved                   |
| Channels          | Full    | Token/webhook configs preserved                    |
| Memory (SQLite)   | Full    | Schema-compatible                                  |
| Memory (Markdown) | Full    | Direct copy                                        |
| Sessions          | Full    | Conversation history preserved                     |
| Custom tools      | Partial | Vtable interface differs; manual adaptation needed |

## Post-migration

After migration, verify your setup:

```bash
seaclaw health              # Check all components
seaclaw agent "Hello"       # Test a simple interaction
```
